## 技术选型指导

### 🎯 Web框架选择决策树

```text
项目规模？
├─ 小型项目（< 10K LOC）
│  ├─ 需要快速开发？ → rocket
│  └─ 需要高性能？ → axum
├─ 中型项目（10K - 100K LOC）
│  ├─ 需要企业级特性？ → actix-web
│  └─ 需要微服务架构？ → tower + axum
└─ 大型项目（> 100K LOC）
   ├─ 需要高并发？ → actix-web
   └─ 需要复杂中间件？ → tower + axum
```

### 🤖 AI集成策略选择

```text
AI功能需求？
├─ 简单文本处理 → 本地处理（candle + wasm）
├─ 复杂推理任务
│  ├─ 实时性要求高？ → 边缘推理 + 云端训练
│  └─ 数据隐私要求高？ → 本地推理
└─ 多模态处理
   ├─ 实时性要求高？ → 边缘多模态推理
   └─ 成本敏感？ → 混合推理策略
```
