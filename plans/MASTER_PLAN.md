# 主计划（可持续推进）

## 愿景

- 以 Rust 为工程主轴，循序落地 AI 系统能力：推理 → RAG → 多代理 → 评测与安全

## 里程碑

- M1 最小可用：本地推理服务（`candle` + `axum`）
- M2 基础 RAG：检索/重排/上下文路由
- M3 工具与执行：代码执行/数据库/工作流
- M4 多代理与监督：角色协作与冲突解决
- M5 评测与安全：过程评测/红队/可观察性

## 机制

- 每 2 周一个 SPRINT；每月回顾调整目标
- 与 `docs/research_watch.md` 联动更新方向
